New release procedure
=====================
1) Update version in src/CMakelist
2) Update files RELEASES CHANGELOG with new version
3) Ejecutar linguist
4) Run distribution.sh to generate distribution file
5) Upload file to sourceforge
6) Update subversion and make tag
7) Make a new ebuild with the new version
8) Upload the build to the subversion portage
9) Update web page
   - Create news object
   - Add release to the page
   - Update links
   - Upload to internet with bash publicar.sh
10) Check links
